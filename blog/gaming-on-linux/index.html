Gaming On Linux

2022-09-01

Gaming on liunx has developed a lot. Also there are lots of user who game on linux. As a linux user I would like to say that it has both advantages and disadvantages. First of all if you can't work for a perticular thing for a long time then linux would not be sutible for you. Also not all the games you will be able to play. It happens the most for latest online games which have anticheat or that kind of stuff in it. For instanse Fortnite doesn't work :'(. Andriod Emulation isn't developed yet so it can't really handle 3D Graphics and some has some architcature issue.

You might familer with Steam but you mightn't heard about lutris. Lutris is a game maneger in simple way. It provides some necessary tweakings,extentions etc. Also there is Wine. Which is the main thing for to run Windows Applications on Linux. It's very necessary application which we all need.

Graphics Drivers.

Specialy for AMD GPUs you don't need to do anything. It is already built in your Linux Kernel. But for Graphics Cards like Nvidia you might need to look around on the internet. Best thing would be look around your distrobution wiki. For me it is Arch Wiki. There are several GPU drivers. I only kow about AMD Drivers. MESA DRIVER it is the best in my opinion and it is open source. Then you got AMDGPU and AMDGPU-PRO and these are from AMD.
