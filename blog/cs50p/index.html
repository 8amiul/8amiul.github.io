I took CS50P

2023-02-04

CS50P is a great course for beginners whose interested in learning Python programming language.

I was also quite interested learning Python. I was looking for a free course. As I don't have money to spend on a course so I had to watch Python tutorials on YouTube or download any course from torrent or FTP server.

I tried learning Python from YouTube and those courses but there was one limitation which is there was no project or work to do.

So those were not that much effective. As the courses were pirated I just had notes and videos. Therefore getting more deeper I found about CS50.

CS50 is a great course from Harvard University and The big thing it's free for everyone. In the CS50 they cover the introduction to computer science. They teach many topics of computer science also they teach C, MySQL, Python and many more programming languages.

CS50 is so large as it's everything about the whole computer science. Then from CS50 there are many other courses such as web, game developing, AI, Python etc. So I then thought why not to take the CS50P which is CS50's Introduction to Programming with Python.

After taking the course I started watching those lectures and submitting those problem sets. That was the most fun part of this course. At first those problem sets were easy. But then many complex part started raising such as Unit Tests and OOP specially. OOP or Object oriented programming did seem a little confusing to me just because of it's weird syntax. Then after looking at some other learning sources I understood many thing about OOP.

On that time I understood that there is no end of a certain programming language. There are a lot of stuff to learn or can be learned. When I was an absolute beginner I just knew or did think that learning python or a programming language is the main aim here then I don't have to learn anything. But it wasn't that correct consideration of mine. In a programming language there are many libraries to learn and by using those we gotta make our own program.

I took the course in Dec 1, 2022 at it end today[Feb 04, 2023].

After submitting the final project they provide me a certificate.

So for the final project I made a terminal based program which does scrape news from a news website called The Daily Star. and I named my program Thedailystar-CLI. A short video about my final project is given below.

I think it would be more better if I could spend more time and effort on it. But I also have some other work to do so I needed to compromise in this regard.

About my program I did use curses to handle key input, Designing and text management. Used BeautifulSoup4 for scraping data from the website. Requests library for sending HTTP request. Also used re library in order to make the width of text subject to the width of terminal.

My final project can be found here.

I really enjoyed CS50P and I would highly suggest to those who are interested in programming to start with Python and take CS50P.
